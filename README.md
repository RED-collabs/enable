# enable
This script is use to , start/ stop services like ssh, tor etc..  
  
#How to use  
step 0: cd ~/Download step 1: Download the bash script 'https://github.com/RED-collabs/enable.git '  
step 2: cd enable  
step 3: type " sudo mv update /usr/local/bin/enable " then type:  
step 4: sudo chmod +x /usr/local/bin/enable "  
if u skip step 4: type " sudo bash enable " to run  
step 5: to run the script type " sudo enable " may ask for usr pass  

sudo enable [services] [status]   
service eg:ssh   
status: 1 = on 
        0 = off
